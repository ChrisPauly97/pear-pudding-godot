## Research Notes

Follow the **Acquisition Instructions** section of `docs/agent/art-sprites.md` (produced by TID-444). In summary:

1. For each slot group, open the primary pick's URL, confirm the license tag on the page matches the doc, and download. Fall back to the backup pick if the license changed or the art doesn't fit once seen at full size.
2. Crop/split sprite sheets into individual PNGs where the manifest calls for single files (the doc will give per-pack instructions; ImageMagick example: `magick sheet.png -crop 16x32+0+0 out.png`).
3. Save to the exact repo paths in the manifest (expected: `assets/textures/characters/`, `assets/textures/props/`, `assets/textures/cards/`).
4. Keep transparent backgrounds intact; do not resample/scale (nearest-filter rendering shows every artifact).
5. While downloading, copy the exact attribution text / author / license / URL from each page — TID-446/447 need it verbatim for the CREDITS file.
6. Commit the PNGs. `.import` sidecars are generated by the editor/headless import later; PNGs need no `.uid` sidecars.

This task is marked done once the files are committed and the licenses/attributions have been captured (paste them into this file's Changes Made section or hand them to the agent session running TID-446).
